# 專區展商 HTML 產生器使用說明

## 1. 工具目的

這個工具的用途是把各專區或國家館提供的展商名單，整理成可以貼到 Drupal 的固定 HTML 卡片。

核心目標不是管理展商資料，而是縮短網站編輯從 Excel 名單到可發布 HTML 的時間，並降低人工整理時常見的錯誤。

## 2. 這個工具解決什麼問題

舊的處理方式通常會遇到：

- 各專區提供的 Excel 格式不一致
- 欄位順序不同，有時甚至混有聯絡人、電話、地址
- 網址沒有 `https://`
- 國家館只有 Company Name / Website，沒有個別 booth
- 編輯者要手動整理成固定 HTML，容易漏資料或貼錯

新的工具把這些工作集中成一個流程：

1. 貼上 Excel 內容
2. 自動解析公司名稱、攤位、網址
3. 在輸出前先檢查缺漏
4. 產生固定格式 HTML

## 3. 舊版與新版對比

### 舊版處理方式

- 直接根據來源資料人工改 HTML
- 欄位錯置時不容易第一時間發現
- 網址格式問題常在貼上後才發現
- 國家館資料若沒有 booth，容易整批卡住

### 新版工具流程

- 可直接貼上較混亂的 Excel 區塊
- 可辨識表頭，不完全依賴固定欄位位置
- 會先顯示檢查狀態，再決定是否輸出
- 國家館可改用共用 booth
- `www.xxx.com` 會自動補成 `https://www.xxx.com`

## 4. 必要資料規則

目前前台卡片一定要有三項資料：

- 公司名稱
- 攤位號碼
- 網站

說明：

- 公司名稱：前台主要辨識資訊
- 攤位號碼：前台主要導覽資訊
- 網站：列為必要資料，但視覺上屬次要資訊

如果是國家館格式，沒有個別 booth 時，請在工具內填入共用攤位號碼。

## 5. 可接受的來源格式

目前至少可處理三類來源：

### A. 已整理好的專區名單

常見欄位：

- `Local Name`
- `Customer_Name`
- `Pavilion Name`
- `Booth Number`
- `Web Address`

這是最理想的格式。

### B. 國家館報名表

常見欄位：

- `No.`
- `Company Name`
- `Contact Person`
- `E-mail`
- `Contact phone`
- `Address`
- `Website`

這類格式通常沒有 booth，需要在工具中填共用 booth。

### C. 主資料庫匯出表

欄位很多，包含訂單、聯絡資訊、地址、會員狀態等。

這類可以作為來源資料，但不建議直接整份原封不動貼入；若直接使用，仍需要先確認工具解析出的欄位是否正確。

## 6. 操作步驟

### 標準專區資料

1. 輸入專區名稱
2. 選擇輸出語系
3. 需要時補專區簡介與重點條列
4. Excel 格式選 `標準格式`
5. 貼上展商名單
6. 按 `檢查並產生 HTML 代碼`
7. 在右側確認：
   - 公司名稱
   - 攤位號碼
   - 網址
8. 預覽無誤後再複製 HTML

### 國家館資料

1. 輸入專區名稱，例如 `Korea Pavilion`
2. 選擇輸出語系
3. Excel 格式選 `國家館表單`
4. 在 `共用攤位` 輸入 booth 號碼
5. 貼上國家館 Excel 區塊
6. 按 `檢查並產生 HTML 代碼`
7. 確認公司名稱與網址都有被正確抓到

## 7. 工具目前的自動處理能力

工具目前會幫忙處理：

- 跳過表頭列
- 辨識國家館常見表頭
- 把 `www.xxx.com` 補成 `https://www.xxx.com`
- 缺個別 booth 時，套用共用 booth
- 在輸出前顯示缺漏或需要檢查的列

## 8. 哪些情況還是要人工確認

- 公司名稱欄位為空
- 攤位號碼缺失
- 網站欄看起來不像網址
- 來源資料沒有表頭
- 原始資料混入太多非展商欄位且格式不一致

## 9. Briefing 用重點

你跟同事 briefing 時，可以直接用下面幾句：

- 這不是資料庫，是把展商名單快速轉成可發布 HTML 的工具。
- 工具已能吃標準專區表，也能處理國家館報名表。
- 前台一定要有公司名稱、攤位號碼、網站。
- 工具會先檢查再輸出，不再是直接人工貼 HTML。
- 國家館若沒有個別 booth，可改填共用 booth。

## 10. 這份工具 / 工作可以怎麼量化

可以用四種方式量化：

### 時間

- 以前整理一個專區名單要多久
- 現在從貼 Excel 到產出 HTML 要多久

### 錯誤率

- 缺 booth
- 網址格式錯誤
- 公司名稱貼錯欄位

### 可處理格式數量

- 標準專區表
- 國家館報名表
- 主資料庫匯出表

### 重工次數

- 以前常需要手動重排 HTML
- 現在改為先在工具裡檢查與修正
