# Tool Optimization Roadmap

This backlog is ordered by user-facing impact and implementation risk.

## Round 1: Product Confidence

1. Sponsors Logo Generator
   - Add first-use empty state and sample data.
   - Add export metadata: project name, export time, tool version.
   - Let preview switch between Chinese and English output.
   - Show missing-link and missing-logo warnings per sponsor.

2. Pavilions Vendor Generator
   - Add parsed-data review table before output.
   - Add row-level validation for missing company name, booth, or URL.
   - Add SEMICON preset color swatches.
   - Add bilingual output instead of single language switching only.

## Round 2: Handover Safety

3. Forum Generator
   - Add JSON project export/import.
   - Preserve parsed forum baseline inside the work file.
   - Show parsed forum review summary before Step 2 editing.
   - Collapse AI settings by default.
   - Add minimum-input mode for simple forum pages.
   - Add Drupal paste checklist on the output step.

4. Theme / Trend Generator
   - Make "AI optional" clearer in the interface.
   - Add empty-field checker.
   - Add quick jump/search for theme blocks.
   - Confirm whether keyword settings should be included in output.

## Round 3: Hub Polish

- Add tool screenshots to the dashboard.
- Add a portfolio/case-study page.
- Add release notes and changelog.
- Normalize terms: use "產生器" consistently in Chinese UI.
- Normalize copy buttons, preview toggles, status labels, and import/export wording.
