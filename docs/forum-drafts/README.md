# Forum Drafts

這裡存放各場論壇的「論壇草稿 JSON」，是 [Forum HTML Compiler](../../tools/forum/README.md) 後續維護的核心資產。

檔名規則：`{論壇代號或關鍵字}-{年份}.json`，例如 `ai-packaging-2026.json`。

**每次用產生器改完一場論壇、貼回 Drupal 之後，立刻匯出草稿 JSON 覆蓋掉這裡對應的舊檔案**，不要等到「有空再存」——這是唯一能讓下一個接手的人不用從 Legacy Restore 重建資料的方式。

合作論壇（不套用我們樣式標準的論壇）不需要在這裡建立草稿 JSON。
